How it works

How Label Mate identifies a record

Getting from a scratched-up label to the right track is a waterfall of decisions — each step narrowing the field, with a fallback ready whenever one comes up empty.

1

Read the label

Apple's Vision framework reads every line of text on the label, building consensus across multiple camera frames so glare or a shaky hand doesn't throw it off — and correcting for common misreads like O / 0 or an upside-down label.

2

Pull out the clues

From that raw text, Label Mate teases apart catalog-number candidates, artist names, and titles — splitting mashed-together fine print and throwing out boilerplate. The catalog number is the strongest signal, so it leads.

3

Look it up

Candidates go to Label Mate's catalog service in a single batched request, matched against 7.5M+ Discogs catalog numbers. Back come the releases each one belongs to, with artist and format attached. Barcodes and album covers feed the same lookup.

4

Pick the right pressing

When one catalog number maps to several pressings, the artist and title read off the label break the tie. If the number turns up nothing, the cascade falls back to resolving straight from artist + title instead.

5

Confirm on Apple Music

Before a note plays, the chosen release is verified against Apple Music. Fuzzy matching handles punctuation, featured artists, and various-artists compilations so you land on the real track — not a re-recording or a lookalike.

6

Drop into the player

The full tracklist loads into the player. Each track streams from Apple Music; when Apple Music doesn't have it, Label Mate drops to a YouTube video, and when only a clip exists you'll still hear a 30-second preview.
